FAQ

How can I get started quickly?

Most users like doing a autoupdate for the world, once daily, so start with that! Type the following into the channel you want to do a update in:  

/autoupdate world 1d

The bot will now send a update for the world, once daily. If you'd like to force it to update at a specific time, run the following command:

/autoupdate update_at <time>

Replace <time> with a UTC time when you want a update at. It MUST be a UTC time, as that's what timezone the bot works in! For example, the following command will do a update at 23:59 UTC: 

/autoupdate update_at 23:59
